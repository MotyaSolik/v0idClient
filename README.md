# v0idClient

A simple cheat client / launcher for [CaveGame](https://github.com/rayshalov/CaveGame).

## Usage

1. Place `CaveGame.exe` in the `source/` folder next to `v0idClient.exe`
2. Launch `v0idClient.exe`
3. Toggle cheats using hotkeys
4. Press **F5** to launch CaveGame with selected cheats active

## Cheats

| Key | Cheat | Description |
|-----|-------|-------------|
| F1 | Noclip | Walk through walls |
| F2 | Godmode | Monsters can't kill you |
| F3 | Vision | Full map visibility, no fog |

## Requirements

- Windows
- .NET 8

## Notes

- Cheats are applied at launch — toggle them **before** pressing F5
- v0idClient is a beta. A full C++/Qt version is planned.
